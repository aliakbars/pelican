Resource URL

POST /event/ticket/create

Description

Create a ticket class for an event

Parameters

name Name of the ticket class
price Price of the ticket
qty Ticket's quota
end_of_sales Format: yyyy-mm-dd
event_id Event ID

Example Response

{
    "status": "success",
    "message": "Tiket baru berhasil ditambahkan!"
}
