Resource URL

POST /event/create

Description

Create an event

Parameters

name Name of the contact person
email Contact person's e-mail
phone 10-12 characters
event_name Name of the event
company_name Name of the company
ticket_quota Allowed value 1-2000
start_date Format yyyy-mm-dd
start_time Format hh:mm
end_date Format yyyy-mm-dd
end_time Format hh:mm
venue Name of the venue

Example Response

{
    "status": "success",
    "message": "Event Anda berhasil disimpan!"
}
