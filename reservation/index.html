Resource URL

POST /reservation/create/{event_id}/verify

Description

Store a reservation

Parameters

name Visitor's name
id_no KTP/SIM/Passport
email Visitor's email
phone Visitor's phone (10-12 characters)
birthdate Format: yyyy-mm-dd
sex M/F
info Where the visitor got the information from
ticket_class_id Ticket class ids
ticket Number of tickets for each ticket class

Example Response

{
    "status": "success",
    "message": "Pemesanan dapat dilakukan"
}
